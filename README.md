# bb8-animation

I hope you'll like it :)

## To RUN:
1. cd to project root
2. `yarn`
3. `yarn run dev`

## To Deploy:
1. cd to project root
2. `yarn`
3. `yarn run deploy`

## Technologies:
* React as SPA library
* GSAP as animation library
* Scss as css preprocessor

## Tested On:
* Firefox 58+
* Chrome 60+
* Edge 18+
